## 目录结构 ##

```pre

├── assets                   // 生产目录(静态资源)
│   ├── css                  // css文件（less文件不用提交给后台）
│   ├── images               // 图片
│   └── js                   // js文件
├── components               // 组件（js形式）
├── node_modules             // 引用的外部框架及插件
├── page                     // html页面文件存放目录

```
